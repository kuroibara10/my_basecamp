## Description

The application allows users to register accounts, log in and out securely, and create and manage projects. Special attention is given to User Experience (UX) to ensure a smooth, intuitive, and user-friendly interface.

The system supports role-based access control, distinguishing between regular users and administrators. Administrators have additional permissions, such as managing user roles, while standard users can focus on project creation and management.

- User Management: User registration, profile display, account creation, and deletion

- Authentication: Secure sign-in and sign-out sessions

- Role Permissions: Assigning and removing admin privileges

- Project Management: Create, view, edit, and delete projects
